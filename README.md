# yt-2-mp3
YouTube To MP3 Converter

To get started
---
1:
Run the YT_2_MP3.exe. The app will open after about 10 seconds.

2:
Copy (CTRL+C) and paste (CTRL+V) a youtube link into the URL box and then hit download. POOF! its done.
